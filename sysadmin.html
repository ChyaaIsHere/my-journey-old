Cara melakukan instalasi Debian 9 pada Virtualbox

Sebelum lanjut ke cara instalasinya, kita harus mengenali apa itu Virtualbox dan Debian.

VirtualBox adalah perangkat lunak virtualisasi yang memungkinkan menjalankan berbagai sistem operasi di satu komputer,
sedangkan Debian adalah distribusi Linux yang dikenal karena stabilitas dan keandalannya, khususnya sebagai sistem operasi server.

Langkah langkah Instalasi :

1. Buka aplikasi VirtualBox, lalu klik [New]

2. Lalu beri nama untuk virtual yang akan kita buat dengan type Debian dan version 64bit, karena saya menggunakan Debian 9 64bit [Next]

3. Tentukan memory yang akan digunakan virtual, disini saya memberikan 4GB atau 4096 MB. Sesuaikan dengan memory di komputer masing-masing lalu [Next]

4. Lalu pada bagian Create a Virtual Harddisk Now sesuaikan dengan kebutuhan kalian. Disini karena hanya simulasi instalasi saja, saya akan menggunakan penyimpanan sebesar 20 GB saja.

5. Pada bagian "Summary", informasi yang tertulis di tab itu adalah informasi mengenai virtual mesin yang kalian buat. Kalian bisa lihat bahwa memory disana terdeteksi 4 GB, storage 20 GB, dan jumlah prosesor yang kalian gunakan.

6. Setelah itu, klik [Finish] untuk menyelesaikan reviewnya.

7. Setelah selesai, kita sekarang lanjut klik [Start]

8. Klik choose a virtual optical disk file

9. Cari file ISO yang kalian download. Sebagai contoh "debian-9.4.0-DVD-1.iso" dan pilih [Open]

10. Lakukan restart mesin virtual dengan kombinasi CTRL kanan + R dan secara otomatis mesin langsung restart.

11. Setelah Debian 9 kita nyalakan akan tampil menu seperti dibawah, dalam tutorial ini saya akan install menggunakan mode text, pilih Install dengan cara tekan [Enter].

12. Pilih Bahasa Indonesia tekan [Enter].

13. Lanjutkan instalasi dengan Bahasa Indonesia yang di pilih lalu klik Ya

14. Pilih lokasinya Indonesia [enter]

15. Pilih keyboard American English tekan [Enter].

16. Untuk konfigurasi ip address, disini saya buatkan 192.168.100.1 [lanjutkan]

17. Untuk konfigurasi hostname saya buat nama KitaBersama, karena kita akan konfigurasi hostname setelah instalasi selesai.

18. Masukkan password untuk user root, saya buat simple aja biar mudah nantinya seperti 123 sampai konfirmasi 123.

19. Saya buat user baru dengan nama seperti kitabersama, boleh gunakan nama lain. Lalu buat Password/kata sandi

20. Untuk time zone silakan dipilih sesuai lokasi tinggal

21. Nah kita masuk ke bagian terpenting dalam instalasi kali ini, yaitu pembagian partisi hardisk. pilih Terpandu - gunakan seluruh harddisk dipartisi debian

22.Selanjutnya akan ditampilkan pilihan tentang konfigurasi partisi yang akan dilakukan. Pilihlah jenis hard disk yang akan dipartisi secara manual. Di sini kita pilih [SCSI3 (0,0,0) (sda) â€“ 20 GB ATA VBOX HARDDISK], walaupun mungkin pada komputer anda ukuran dan jenis hard disk bisa saja berbeda. Lalu tekan Enter.

23. Selanjutnya kita akan ditanya bagimana skema partisi untuk instalannya. Karena saya juga belum memahami skema partisinya kita Pilih semua berkas di satu partisi (disarankan untuk pemula)

24. Setalah kita yakin dengan pembagian pastisinya pilih selesai mempartisi dan tulis perubahan-perubahannya ke harddisk [Enter] di jendela selanjutnya pilih Ya

25. Sistem akan meminta untuk men-scan DVD Debian lainnya. Jika ada ISO DVD2, DVD3, dll., mount dari VirtualBox terlebih dahulu. Ini menambahkan repositori DVD ke file list Debian. Untuk saat ini, pilih [Tidak] dan tekan Enter.

26. Selanjutnya, instalasi akan menanyakan apakah Anda ingin menginstal paket tambahan melalui jaringan internet (network mirror). Karena PC Anda tidak terhubung ke internet, pilih [Tidak] dan tekan Enter.

27. Kemudian, proses instalasi meminta kesediaan kita untuk berpartisipasi dalam survey paket Debian. Kita tidak perlu berpartisipasi, maka pilihlah [Tidak], lalu tekan Enter.

28. Pilih "Standard system utilities" dari daftar aplikasi. Gunakan SPASI untuk menghilangkan tanda * pada paket yang tidak diinginkan. disini saya mau menggunakan tampilan CLI. jadi yang perlu dilakukan adalah menghapus tanda * pada opsi "Debian desktop environment" lalu tekan [Enter].

29. Pada jendela berikutnya, proses instalasi akan meminta untuk menginstal GRUB boot loader pada Master Boot Record. GRUB adalah program yang mem-boot sistem operasi dari hard disk atau media lainnya. Pilih [Ya] dan tekan Enter untuk menginstal GRUB.

30. Lalu pilih hardisk mana yang nanti akan di taruh GRUB loader. Saya pilih /dev/sda lalu [Enter]

31. Setelah selesai akan diinformasikan bahwa proses instalasi telah selesai. Pilih [lajutkan] lalu [Enter].Debian akan secara otomatis Reboot.

Finish! Kamu telah menyelesaikan tahap instalasi debian pada virtualbox!

Cara melakukan remote jarak jauh menggunakan aplikasi PuTTY!

Sebelum lanjut ke cara instalasinya, kita harus mengenali terlebih dahulu apa itu PuTTY?

PuTTY adalah aplikasi klien SSH yang memungkinkan akses remote ke server dengan protokol aman seperti SSH, Telnet, dan rlogin, dan tersedia untuk Windows serta platform lain.

Langkah langkah untuk Remote Debian menggunakan PuTTY :

1. Lakukan cek konektivitas terhadap kedua device (debian dan device client).

2. Jika keduanya masih bisa saling reply, maka kita bisa lanjut ke cara selanjutnya.

3. Masukkan file ISO kedalam Optical Drive atau jika kalian melakukannya di laptop asli, kalian bisa pasang kembali flashdisk yang berisikan bootable ke laptop/komputernya.

4. masukan perintah di terminal debian "apt-cdrom add" dan enter.

5. Pada bagian ini, kalian tunggu sampai terbaca linux debian yang kalian masukkan tadi. dan jika sudah terdeteksi, kita bisa lanjut ke tahap selanjutnya

6. Kita akan lakukan instalasi aplikasi Openssh-server. Jadi, kita perlu memasukan perintah instal "apt install openssh-server" lalu enter.

7. Setelah itu kalian tekan tombol y untuk konfirmasi apakah ingin di install atau tidak. dan proses instalasi akan berlangsung.

8. Setelah aplikasi openssh sudah berhasil di install, yang perlu di lakukan adalah melakukan konfigurasi terhadap ssh yang baru di instal.

9. Tahap selanjutnya, masukan perintah "nano /etc/ssh/sshd_config".

10. Didalam sana, yang perlu di rubah adalah bagian port. Hapus tanda pagar di depan teks port dan berikan port yang kalian mau. sebagai contoh dari 22 saya rubah menjadi 2024.

11. Lalu scroll kebawah dengan tombol anak panah kebawah dan cari "#PermitRootLogin prohibit-password".

12. Jika kalian ingin masuk kedalam user root lewat ssh, kalian hapus tanda # dan ubah tulisan "prohibit-password" menjadi "yes".

13. Setelah itu kalian bisa lakukan save file confignya dengan cara tekan tombol dengan kombinasi CTRL + X, tekan y, dan ENTER. Maka, dengan begitu file ssh sudah selesai kita konfigurasi.

14. Setelah tahap tadi, kita harus lakukan restart ssh service terlebih dahulu agar konfigurasi yang kita lakukan tadi langsung diterapkan sistem. dengan memasukkan perintah "/etc/init.d/ssh restart" atau "systemctl ssh restart".

15. Buka aplikasi PuTTY dan masukan hostname beserta portnya. Untuk hostname saya masukan 2.2.2.1 (IP ADDRESS DEBIAN) dan port 2024.

16. Tekan "Open" dan akan terbuka tab baru dengan bertuliskan "PuTTY Security Alert" disini kalian tekan "Accept".

17. Kamu bisa coba untuk masuk kedalam user root dan user biasa di PuTTY.

Bagaimana cara membuktikannya jika PuTTY berhasil terhubung ke Debian yang saya jalankan?

Kalian bisa coba untuk melakukan shutdown atau reboot dengan perintah yang sama seperti di debian.

Finish! Kamu telah menyelesaikan tahap remote debian pada PuTTY!
